# Phase 1: Setup & credentials — your actions

Do these in the [Snap Developer Portal](https://developers.snap.com) before or in parallel with Phase 2.

- [ ] **1.1** Create a Camera Kit App (if not already created).
- [ ] **1.2** Generate API Token; store it securely (you’ll add it to config in Phase 2).
- [ ] **1.3** Add allowed domain: `http://localhost:3000` (exact).
- [ ] **1.4** Get your published Lens: copy **lensId** and **lensGroupId** from Lens Repository.

**Exit criteria:** You have the API token and both lens IDs ready to paste into `src/config/snapConfig.ts`.
